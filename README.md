# Bank - Java Swing Banking Application

A simple desktop banking application built using core Java and Swing UI.  
This project demonstrates object-oriented programming, event-driven UI, and transaction handling.

---

## Features

- Deposit money
- Withdraw money
- View current balance
- Transaction history tracking
- Basic error handling (e.g., insufficient balance)

---

## 📁 Project Structure

Bank/
│── Main.java
│── Account.java
│── Transaction.java
│── BankService.java
│── BankUI.java

---

## Concepts Covered

- Object-Oriented Programming (OOP)
  - Classes & Objects
  - Encapsulation
- Collections (`ArrayList`) for storing transactions
- Exception Handling (runtime validation)
- Java Swing (GUI development)
- Event Handling (button clicks)
- Date & Time API (`LocalDateTime`)

---

## UI Overview

- Input field for amount
- Deposit / Withdraw buttons
- Transaction history popup
- Balance display

---

   cd Bank


# TaskManager - Java Swing Task Management App

A simple task manager desktop application built using Java Swing.  
It allows users to create, track, and manage tasks with priorities.

---

## Features

- Add new tasks
- Assign priority (High / Medium / Low)
- Mark tasks as complete
- View all tasks in a list
- Error handling for invalid input

---

## 📁 Project Structure

TaskManager/
│── Main.java
│── Task.java
│── TaskService.java
│── TaskUI.java

---

## Concepts Covered

- Object-Oriented Programming (OOP)
  - Class design
  - Encapsulation
- Collections (`ArrayList`) for task storage
- Java Swing (GUI development)
- Event-driven programming
- Exception handling
- MVC-like separation (UI + Service + Model)

---

## UI Overview

- Input field for task title
- Dropdown for priority
- Add Task button
- Mark Complete button
- Task list display
